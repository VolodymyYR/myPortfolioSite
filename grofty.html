Growfy

 --- Link to the layout of this project ---

This responsive landing page for a marketing agency was developed using modern web development approaches. The foundation is built on semantic HTML markup with a clear block structure that enhances content accessibility and supports SEO optimization. For styling, modular CSS with SASS preprocessor was implemented, allowing for logical organization of styles and avoiding code duplication.

Responsiveness was achieved through a mobile-first approach using media queries and a flexible grid system based on Flexbox and CSS Grid. A distinctive feature is the pixel-perfect layout reproduction, accomplished through careful work with dimensions, spacing, and typography. All sizes are specified in relative units (rem, %) to ensure proper scaling across different devices.

The JavaScript functionality was implemented using pure Vanilla JS without reliance on third-party libraries. Core features include modal windows, a mobile burger menu, and smooth scroll effects. The code follows a modular structure with clear separation of concerns, where individual functions handle specific tasks, improving readability and maintainability.

Image optimization was implemented using srcset for different screen resolutions, enhancing loading speed. Forms include basic validation through HTML5 attributes and additional JavaScript checks. Styled states of interactive elements (:hover, :focus, :active) add interactivity without compromising performance.

This project demonstrates the ability to work with complex adaptive layouts, write clean and structured code, and utilize modern frontend development technologies. The absence of unnecessary dependencies highlights a strong understanding of native technologies and the capability to create efficient solutions with a minimal toolset.
