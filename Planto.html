Adaptive Landing Page for Indoor Plant Shop

This adaptive landing page for an indoor plant store showcases my skills in pure HTML, CSS (using SCSS), and JavaScript, along with third-party library integration. Developed with strict adherence to the provided design mockup, it demonstrates my ability to precisely implement visual concepts. The color palette, typography, spacing, and overall composition perfectly match the original layout, including pixel-perfect rendering of key sections.

The project utilizes a modern tech stack: semantic HTML5 with proper use of 'section', 'article', 'header', and 'footer' tags; SCSS for efficient style management with nested rules, variables for colors and fonts, mixins for reusable elements, and well-structured media queries ensuring full responsiveness. All interactivity is implemented in vanilla JavaScript without jQuery dependencies, highlighting my understanding of core language principles.

The display adapts flawlessly across devices thanks to carefully planned breakpoints and a flexible structure. For product sliders and customer testimonials, I integrated Swiper.js?v=1746853878812 with customized navigation, pagination, and autoplay settings. Additionally, I implemented various interactive elements including animated buttons, a modal feedback form, and an adaptive burger menu for mobile devices.

Special attention was given to performance optimization: images were compressed for fast loading, CSS and JS were minified for production, and lazy loading was applied to certain media resources. The subtle animations, smooth transitions, and thoughtful timing functions significantly enhance user experience.

This project demonstrates my capabilities as a frontend developer: the ability to work with design mockups, create responsive websites without CSS frameworks, integrate third-party solutions, and optimize code for better performance while maintaining clean, semantic markup and delivering polished visual details. The implementation balances aesthetic appeal with technical precision, using modern development practices to create a production-ready, user-friendly interface.
