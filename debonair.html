Debonaire

 --- Link to the layout of this project ---

The Debonaire website is a responsive landing page for a barbershop, built with a focus on clean code, performance, and precise layout implementation. The project utilizes semantic HTML5, modern CSS3 with Flexbox and Grid, and vanilla JavaScript for interactive elements, avoiding unnecessary dependencies. The structure follows a logical flow, with clearly defined sections for services, testimonials, and contact information, all styled with maintainable and scalable CSS.

A key feature is the mobile-first responsive design, which smoothly adapts to various screen sizes, from smartphones to large desktop displays. Media queries ensure optimal typography, spacing, and image scaling, while maintaining visual consistency with the original design. The navigation menu transforms into a hamburger menu on smaller screens, implemented with a lightweight JavaScript toggle for smooth opening and closing without page jumps.

CSS follows BEM (Block Element Modifier) methodology for improved readability and reduced specificity conflicts. Reusable utility classes and CSS variables streamline styling, making future updates efficient. The gallery section leverages CSS Grid for a structured yet flexible layout, automatically adjusting image placement based on available space.

JavaScript enhances user interaction with a smooth scroll effect for anchor links and dynamic behavior for the testimonial slider. Event delegation optimizes performance by minimizing listeners, while the slider functionality is built with manual controls for better user control. Form inputs include basic validation, ensuring usability without relying on external libraries.

Performance optimizations include optimized image assets, semantic markup for SEO, and efficient asset loading. The codebase demonstrates strong attention to detail in matching the design mockup while ensuring maintainability and scalability. Cross-browser compatibility is maintained through progressive enhancement, ensuring accessibility and functionality across different devices and browsers. This project highlights strong frontend development skills with an emphasis on clean, efficient, and adaptable code.
