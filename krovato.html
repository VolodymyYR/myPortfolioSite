Krovato

Krovato is a modern adaptive landing page for a furniture company, created during a web development course. The project demonstrates the practical application of acquired knowledge, focusing on clean code, precise layout implementation, and the use of modern frontend technologies.

The site is built with semantic HTML markup, enhancing accessibility and SEO optimization. For styling, the BEM methodology is used, ensuring modularity and easy code maintenance. CSS is written using the SCSS preprocessor, allowing for variables, mixins, and nested selectors, which simplifies development and style management.

Responsiveness is achieved through media queries, ensuring correct display on all types of devices, from smartphones to wide-screen monitors. Smooth interactive elements, such as hover effects and animations, are implemented using CSS transitions and transforms.

Special attention is given to JavaScript: a custom testimonial slider is built without relying on third-party libraries, demonstrating a solid understanding of DOM manipulation and event handling. The code follows the DRY (Donâ€™t Repeat Yourself) principle, making it easy to extend and modify.

Cross-browser compatibility is ensured with Autoprefixer, and all images are optimized for fast loading. The use of SVG icons instead of raster graphics improves visual quality on high-density screens.

The project fully adheres to the provided layout, including precise spacing, color schemes, and typography. Working on this site during the course helped solidify key frontend development skills, such as markup, working with preprocessors, responsive design, and vanilla JavaScript. The result is a high-quality, optimized product ready for real-world use.
