HomeDecor

 --- Link to the layout of this project ---

The HomeDecor website is a responsive single-page application showcasing home decoration products with a clean, modern interface. Built with HTML5, CSS3, and vanilla JavaScript, the project demonstrates strong frontend development skills with a focus on performance and maintainability. The layout follows a mobile-first approach, ensuring seamless responsiveness across devices from 320px to 1920px viewports. CSS is organized using BEM methodology, with variables for consistent colors and breakpoints, while JavaScript implements modular patterns for product filtering functionality.

The product filtering system is implemented through efficient DOM manipulation and event delegation, avoiding jQuery dependencies. Products are dynamically filtered based on category selection without page reloads, demonstrating solid JavaScript fundamentals. The gallery section uses CSS Grid for responsive image display, adapting column counts based on viewport size. Form validation includes proper input types and required attributes, with potential for enhanced client-side validation.

Performance optimizations include optimized image assets, semantic HTML5 markup for better accessibility, and clean CSS with minimal redundancy. The codebase shows attention to detail in matching the original design mockups while maintaining flexibility for content updates. Cross-browser compatibility is ensured through vendor prefixing and progressive enhancement techniques. The project exemplifies modern frontend development practices with a focus on clean, maintainable code and responsive design principles suitable for portfolio demonstration.
