Thesis

This project is a graduation thesis website built using modern web technologies with a focus on clean code, responsiveness, and semantic markup. The foundation is built on **HTML5** with a clear structure and **CSS3** utilizing the **BEM methodology** to ensure modularity and easy code scalability. Full responsiveness is achieved through **media queries**, guaranteeing proper content display across all device types – from widescreen monitors to mobile phones.

"JavaScript" is used to implement dynamic elements such as a mobile-friendly burger menu and interactive components. The code follows **clean code principles**: functions have logical names, modular approaches are employed, and the global namespace remains unpolluted. To optimize the development process, **Parcel** is used for bundling, automating tasks like minification, code transpilation, and dependency management.

The website strictly adheres to the layout's technical requirements, with precise spacing, typography, and color schemes. Performance is optimized through **image compression**, modern formats (e.g., **WebP**), and minimized CSS/JS. Animations are implemented via **CSS transitions**, ensuring smooth interactivity without excessive browser load.

This project demonstrates the ability to create **responsive, technically refined websites** using modern web development tools while maintaining high standards of code quality and user experience.

Key Features:

  • Semantic HTML5 & CSS3 (BEM)** – Clean, scalable, and maintainable
  • Fully Responsive** – Media queries for all screen sizes
  • Optimized Performance** – WebP images, minified assets
  • Modern Tooling** – Parcel for automated bundling
  • CSS Transitions** – Smooth, lightweight animations
  • Pixel-Perfect** – Precise spacing and typography
  • SEO-Friendly** – Semantic structure and accessibility

The project showcases expertise in **adaptive web development**, balancing technical excellence with user-centric design.
